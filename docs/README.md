# Zync - Personal Dashboard

## Project Overview

Zync is a personal dashboard that integrates health/fitness data with enhanced search functionality, providing users with a unified view of their personal data.

## Core Objectives

- **Garmin Integration**: Connect to Garmin Connect to fetch health and fitness data
- **Authentication**: Secure user authentication and authorization  
- **Enhanced Search**: Advanced search functionality with filtering and data discovery
- **Modern UI**: Clean, responsive interface for data exploration
- **Extensible Architecture**: Built to support future integrations

## Initial Target

1. Integrate Garmin data (activities, health metrics)
2. Provide better search functionality than native Garmin interface
3. Secure user authentication
4. Foundation for adding more integrations in the future

## Future Vision

Expand into a general-purpose personal dashboard aggregating data from multiple sources (calendars, notes, financial data, etc.) with intelligent search and analytics.

---

*Starting simple with Garmin integration and search, building a foundation for a comprehensive personal data hub.*
